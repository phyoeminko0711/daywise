# 🌞 DayWise

DayWise is a modern Flutter application that provides daily guidance based on the Myanmar Calendar and Gregorian Calendar.

Users can select any date and instantly view traditional daily recommendations, including do's and don'ts, auspicious information, and calendar insights.

## Features

- 📅 Gregorian Calendar
- 🇲🇲 Myanmar Calendar
- 🌞 Daily Guidance
- ✅ Do's
- ❌ Don'ts
- ⭐ Auspicious Day Information
- 🔔 Daily Reminder (Future)
- ❤️ Favorite Dates (Future)
- 👑 Premium Features (Future)
- 🌙 Light & Dark Theme (Future)
